Route Preview

Displays the destination points of the current route graphically. The valid movement zone is denoted by a dark gray dashed line. Points can be manipulated using either the mouse or keyboard.

  • Click a point to select it.
  • Click and drag a point to move it.
  • Use the arrow keys to move the selected point.
  • Double click a point, or press enter on the selected point to duplicate it.

Points destinations are adjusted to lie along a square grid of the size specified by the "Snap to grid" input box.

NOTE: All coordinates are depicted as offset from bottom-right of movement zone, with x-axis lying vertically. Ensure CNC machine is in this position before sending route.
